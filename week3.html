🎭 Haar Cascade — Face Detection Demo (คลิกเพื่อสาธิต)
python · Workshop 3-1 · Haar Cascade Face Detection
import cv2

# โหลด pre-trained Haar Cascade (มีใน OpenCV)
face_cascade = cv2.CascadeClassifier(
    cv2.data.haarcascades + 'haarcascade_frontalface_default.xml'
)

# โหลดภาพและแปลงเป็น grayscale
img  = cv2.imread('image.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# ตรวจจับใบหน้า
faces = face_cascade.detectMultiScale(
    gray,
    scaleFactor=1.1,    # ลดขนาด 10% ต่อรอบ
    minNeighbors=5,    # bbox ซ้อนกันน้อยสุด 5 ชิ้น
    minSize=(30, 30)
)

# วาดกรอบ
for (x, y, w, h) in faces:
    cv2.rectangle(img, (x, y), (x+w, y+h), (0, 255, 0), 2)

print(f"พบใบหน้า: {len(faces)} ใบหน้า")
📌
scaleFactor: ยิ่งใกล้ 1 ยิ่งแม่น แต่ช้ากว่า (1.1 = ลด 10% ต่อรอบ)
📌
minNeighbors: สูงขึ้น = False Positive น้อยลง แต่อาจพลาดใบหน้า
📌
ต้อง grayscale ก่อน เสมอ — Haar ทำงานกับ intensity ไม่ใช่สี
🧠 Deep Learning

Deep Learning Methods (2013–ปัจจุบัน)

แบ่งเป็น 2 กลุ่มหลัก: Single-stage vs Two-stage

⚡ Single-Stage (YOLO)
🎯 Two-Stage (R-CNN)
⚡
Single-Stage Detector
ประมวลผลทั้งภาพครั้งเดียว → output bounding box + class ในขั้นตอนเดียว

ตัวอย่าง: YOLO, SSD, RetinaNet
จุดแข็ง: เร็วมาก เหมาะ real-time
จุดอ่อน: อาจแม่นน้อยกว่า two-stage บน object เล็ก
Single-Pass → Direct Output
🎯
Two-Stage Detector
Stage 1: Region Proposal Network (RPN) หา "น่าจะมีวัตถุที่นี่"
Stage 2: Classify + refine bbox แต่ละ region

ตัวอย่าง: Faster R-CNN, Mask R-CNN
จุดแข็ง: แม่นยำสูง
จุดอ่อน: ช้ากว่า single-stage
RPN → Classify → Refine
⚡
YOLO เลือก Single-Stage เพราะต้องการ real-time — YOLO11 ทำได้ 200+ FPS บน GPU สมัยใหม่
⚡ YOLO

YOLO: You Only Look Once

วิวัฒนาการจาก v1 (2016) ถึง YOLO26 (2026) — ทศวรรษแห่งการพัฒนา

2016
YOLOv1 — ปฏิวัติวงการ
แนวคิด "look once" — แบ่งภาพเป็น grid, predict bbox + class ในเซลเดียว, 45 FPS real-time
2018
YOLOv3 — Multi-scale Detection
Detect 3 scale, Darknet-53, detect วัตถุเล็กได้ดีขึ้นมาก
2020–2023
YOLOv5 → YOLOv8 — Ultralytics Era
PyTorch-based, API ดี, YOLOv8 เป็น anchor-free ครั้งแรก รองรับ detect+seg+pose
Sep 2024
YOLO11 — Current Stable ✅ สอนใน PPTX
C3k2 blocks + C2PSA attention, parameter น้อยกว่า YOLOv8m ถึง 22% แต่ mAP สูงกว่า, รองรับ 6 tasks
Jan 2026
YOLO26 — Latest 🆕 NEW
NMS-free inference, edge deployment optimized, open-vocabulary support, MuSGD optimizer — latest production model

🔬 How it Works

How YOLO Work?

คลิก cell ในภาพเพื่อดูว่า YOLO "คิด" อะไรในแต่ละตำแหน่ง

YOLO แบ่งภาพเป็น S×S grid · คลิกเพื่อดูรายละเอียด

📦 Output per Cell

← คลิก grid cell เพื่อดู
output vector ของแต่ละ cell
💡
ถ้า object center ตกใน cell ไหน cell นั้นรับผิดชอบ predict วัตถุนั้น
Anchor-based vs Anchor-free YOLO11 ใช้ Anchor-free

YOLO11 เปลี่ยนมาใช้ Anchor-free ทำให้ง่ายขึ้น แม่นขึ้น

❌ Anchor-based (เก่า)
Pre-define "anchor boxes" หลายขนาด → predict offset จาก anchor — ต้องเลือก anchor size ให้เหมาะกับ dataset, ซับซ้อน
[tx, ty, tw, th, conf, c1..cn]
× จำนวน anchors × cells
✅ Anchor-free (YOLO11)
Predict center point + size โดยตรง ไม่มี anchor — ง่ายกว่า generalize ดีกว่า ไม่ต้อง tune anchor
[cx, cy, w, h, conf, c1..cn]
predict โดยตรงต่อ cell

python · Workshop 3-3 · Complete Detection Code
from ultralytics import YOLO
import cv2
from matplotlib import pyplot as plt

# ─── โหลดโมเดล ───
model = YOLO('yolo11n.pt')  # n=nano, s=small, m=medium, l=large, x=xlarge

# ─── Detect ───
results = model(
    'image.jpg',
    conf=0.35,       # confidence threshold
    iou=0.45,        # NMS IoU threshold
    classes=[0, 2],  # 0=person, 2=car (ไม่ระบุ = detect ทุก class)
    imgsz=640,
)

# ─── อ่านผลลัพธ์ ───
result = results[0]
for box in result.boxes:
    x1, y1, x2, y2 = box.xyxy[0]    # พิกัด bbox
    conf  = float(box.conf[0])      # confidence score
    cls   = int(box.cls[0])         # class index
    label = model.names[cls]           # ชื่อ class
    print(f"{label}: {conf:.2%} at [{x1:.0f},{y1:.0f},{x2:.0f},{y2:.0f}]")

# ─── แสดงภาพผล (สำหรับ Colab) ───
img_result = result.plot()   # วาด bbox บนภาพ
img_rgb    = img_result[..., ::-1]  # BGR → RGB

plt.figure(figsize=(12, 8))
plt.imshow(img_rgb)
plt.axis('off')
plt.title(f'YOLO11 Detection — {len(result.boxes)} objects found')
plt.show()

# ════════════════════════════════════════════
#  YOLO11 Object Detection — Google Colab
#  JOB 3 · 31906-2016 Object Detection
# ════════════════════════════════════════════

# ─── 1. Runtime → Change runtime type → T4 GPU ───

# ─── 2. ติดตั้ง ───
!pip install ultralytics -q

# ─── 3. Import ───
from ultralytics import YOLO
import cv2, numpy as np
from matplotlib import pyplot as plt
from google.colab import files

# ─── 4. อัปโหลดภาพ ───
uploaded = files.upload()
filename = list(uploaded.keys())[0]

# ─── 5. โหลดโมเดล ───
model = YOLO('yolo11n.pt')   # โหลดอัตโนมัติครั้งแรก

# ─── 6. Detect ───
results = model(filename, conf=0.35, imgsz=640)
result  = results[0]

# ─── 7. แสดงผล ───
print(f"พบวัตถุ: {len(result.boxes)} ชิ้น")
for box in result.boxes:
    label = model.names[int(box.cls)]
    conf  = float(box.conf)
    print(f"  ✓ {label:20s} — {conf:.2%}")

img_out = result.plot()[..., ::-1]
plt.figure(figsize=(12,8))
plt.imshow(img_out); plt.axis('off')
plt.title(f'YOLO11 — {len(result.boxes)} objects')
plt.tight_layout(); plt.show()

# ─── 8. Save ผลลัพธ์ ───
cv2.imwrite('result.jpg', result.plot())
files.download('result.jpg')
