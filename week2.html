🔵 โครงสร้างภาพดิจิทัล

ภาพดิจิทัลคืออะไร?

ภาพ = ตารางของ Pixel แต่ละ Pixel มีค่าสีที่กำหนดได้

🎨 RGB Color
⬛ Grayscale
💡
ภาพสี RGB แต่ละ Pixel มี 3 ช่อง (Channel): Red, Green, Blue — แต่ละช่องมีค่า 0–255 รวมเป็น array ขนาด (H × W × 3) — ⚠️ OpenCV ใช้ลำดับ BGR ไม่ใช่ RGB!
🔴
Red
B=200, G=0, R=0
🟢
Green
B=0, G=180, R=0
🔵
Blue
B=220, G=0, R=0
🎨
Mixed
R=200, G=180, B=220
คลิก pixel เพื่อดูค่า
📍 ค่า Pixel ที่เลือก
คลิกที่ pixel ใดก็ได้ →
🔢
OpenCV เก็บเป็น NumPy Array
img.shape = (H, W, 3)
💡
Grayscale แต่ละ Pixel มีค่า เดียว 0–255 (0=ดำสนิท, 255=ขาวสนิท) เก็บเป็น array ขนาด (H × W) — ใช้หน่วยความจำน้อยกว่า RGB ถึง 3 เท่า
0 (ดำ)64128 (เทากลาง)192255 (ขาว)
ลาก slider เพื่อเปลี่ยนค่า grayscale
ค่า: 128
แปลง RGB → Grayscale: Gray = 0.299R + 0.587G + 0.114B
📐 ความละเอียดภาพ (Resolution)

Resolution = จำนวน Pixel กว้าง × สูง — ยิ่งมากยิ่งคมชัด แต่ไฟล์ใหญ่ขึ้น

📌
ใน OpenCV: img.shape[1] = กว้าง (Width/Columns), img.shape[0] = สูง (Height/Rows)

🌀 Filters

Filter พื้นฐาน

กดเพื่อดูผลลัพธ์ Filter แต่ละแบบแบบ real-time

🌫️ Blur 3 แบบ
🔵
Gaussian Blur
ใช้บ่อยที่สุด ลดสัญญาณรบกวนได้ดี ก่อนทำ Edge Detection
cv2.GaussianBlur(img, (5,5), 0)
📦
Box Blur
ค่าเฉลี่ยของ pixel รอบข้าง เร็วกว่า Gaussian แต่ผลลัพธ์หยาบกว่า
cv2.blur(img, (5,5))
🔴
Median Blur
ใช้ค่ากลาง ดีมากสำหรับลด Salt & Pepper noise รักษาขอบได้ดี
cv2.medianBlur(img, 5)
🎨 เปลี่ยนโหมดสี + ⚡ Canny Edge
🎨
cvtColor()
แปลงโหมดสีต่าง ๆ
# BGR → RGB (Matplotlib)
cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
# BGR → Gray
cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
# BGR → HSV
cv2.cvtColor(img, cv2.COLOR_BGR2HSV)
⚡
Canny Edge Detection
หาขอบ (edge) ในภาพ ต้องแปลงเป็น Grayscale ก่อน แล้ว Blur เพื่อลด noise
# ขั้นตอน: Gray → Blur → Canny
gray = cv2.cvtColor(img,
         cv2.COLOR_BGR2GRAY)
blurred = cv2.GaussianBlur(
         gray, (5,5), 0)
edges = cv2.Canny(blurred,
         100, 200)

# 2) Import
import cv2
import numpy as np
from matplotlib import pyplot as plt
from google.colab import files

# 3) อัปโหลดภาพ
uploaded = files.upload()
filename = list(uploaded.keys())[0]

# 4) โหลดภาพ
img = cv2.imread(filename)
img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)

# 5) แสดงโครงสร้าง
print(f"Shape: {img.shape}")
print(f"H={img.shape[0]}, W={img.shape[1]}, C={img.shape[2]}")

# 6) แสดงภาพ
plt.figure(figsize=(10, 4))

plt.subplot(1, 3, 1)
plt.imshow(img_rgb); plt.title('Original'); plt.axis('off')

plt.subplot(1, 3, 2)
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
plt.imshow(gray, cmap='gray'); plt.title('Grayscale'); plt.axis('off')

plt.subplot(1, 3, 3)
blurred = cv2.GaussianBlur(gray, (5,5), 0)
edges = cv2.Canny(blurred, 100, 200)
plt.imshow(edges, cmap='gray'); plt.title('Canny Edge'); plt.axis('off')

plt.tight_layout(); plt.show()
