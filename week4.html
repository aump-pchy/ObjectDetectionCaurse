🏷️ Annotation

YOLO Annotation Format

วาด bounding box แล้วบันทึกเป็นตัวเลข — ทดลองวาดด้านล่างได้เลย

รูปแบบข้อมูล: 1 บรรทัด = 1 วัตถุ
class_id
·
cx
·
cy
·
w
·
h
class_id — หมายเลข class (0, 1, 2...)
cx, cy — จุดกึ่งกลาง bbox (normalized 0–1)
w, h — กว้าง/สูง bbox (normalized 0–1)
⚠️ Normalized Coordinates — ไม่ใช่ pixel!

ทุกค่าหารด้วย ความกว้าง/ความสูงของภาพ → ได้ค่า 0–1 — ทำให้ใช้ได้กับทุกขนาดภาพ

ภาพขนาด 280×200 px
← ภาพขนาด 280×200 px
bbox สีเขียวอยู่ตรงกลาง

 class_id = 0 (พิชิตาโขน)
cx = 140 ÷ 280 = 0.500
cy = 90 ÷ 200 = 0.450
w = 100 ÷ 280 = 0.357
h = 120 ÷ 200 = 0.600

บันทึกเป็น:
0 0.500 0.450 0.357 0.600

📊 Data Split

การแบ่ง Dataset

แบ่งข้อมูลให้ถูกต้อง — โมเดลถึงจะ generalize ได้ดี

Train Set
ใช้ฝึกโมเดล — ต้องมี annotation ครบทุกภาพ โมเดลจะ learn pattern จากชุดนี้
Validation Set
ใช้ตรวจสอบระหว่างเทรน — เปรียบ predict กับ annotation จับ overfitting
Test Set
ใช้ทดสอบสุดท้าย — ไม่มีผลต่อการเทรน วัดผลจริงของโมเดล
เลือกดู split ที่เหมาะกับ dataset ของคุณ
📦 ข้อมูลน้อย
📚 ข้อมูลมาก
Train 80%
Test 20%
Train — 80%
Val — 0%
Test — 20%
📦 ข้อมูลน้อย (แนะนำ 80/20) — เมื่อมีภาพน้อยกว่า ~500 ใบ ใช้ test set ทำหน้าที่ validate ด้วย — ผลอาจไม่นิ่งนักเพราะ sample น้อย
⚠️
ข้อควรระวัง: อย่าเอาภาพเดิมซ้ำกันใน train และ test set — จะทำให้ mAP สูงแบบหลอก (data leakage)
🔀 Augmentation

Data Augmentation

เพิ่ม diversity โดยไม่ถ่ายภาพเพิ่ม — ป้องกัน Overfitting

🧪
ทำไมต้อง Augmentation?
  • เพิ่มจำนวน dataset โดยไม่ต้องเก็บภาพเพิ่ม
  • ให้โมเดล generalize ได้กับสภาพแสง มุม ขนาดต่างๆ
  • ลดปัญหา Overfitting
  • เพิ่มความ robust ต่อ noise และการหมุน

🤖 Roboflow

Roboflow — All-in-One Platform

Upload → Label → Augment → Train → Deploy — จบในที่เดียว

🚀
Roboflow เป็น platform ที่ครบที่สุดสำหรับ computer vision dataset — ฟรีสำหรับ public project, มี Universe (50,000+ datasets), และรองรับ YOLO11 / YOLO26
🔄 Workflow: ตั้งแต่ภาพจนถึงโมเดล
📁
1. สร้าง Project + Upload ภาพ
เปิด roboflow.com → New Project → เลือก Object Detection → drag & drop รูปภาพได้เลย รองรับ .jpg .png .mp4
roboflow.com
🏷️
2. Annotate — วาด Bounding Box
ใช้ Roboflow Annotate วาด bbox ด้วยมือ หรือ Label Assist 🪄 ให้ AI suggest annotation อัตโนมัติ แก้ตรงไหนไม่ถูกก็แก้ได้
✨ Label Assist NEW
📊
3. Generate Dataset Version
กด Generate → เลือก split ratio (70/20/10 หรือ 80/20) → เพิ่ม Augmentation → Roboflow สร้าง dataset version ให้
Augmentation built-in
🏥
4. Health Check — ตรวจคุณภาพ dataset
Roboflow วิเคราะห์ class imbalance, annotation heatmap, image size distribution — บอกว่า dataset ดีพอยัง
Dataset Health
⚡
5. Train บน Roboflow
กด Train → เลือก YOLO11 (หรือ YOLO26) → เลือก pretrained COCO → ตั้ง epoch → รอผล — ไม่ต้อง GPU เอง!
YOLO11 / YOLO26 supported
🚀
6. Deploy + ทดสอบ
ดาวน์โหลด .pt weights หรือใช้ Roboflow API สำหรับ inference — ทดสอบผ่าน browser ได้ทันที
Export YOLO / ONNX / TFLite
🤖 เปิด Roboflow

JOB 4 — ส่งจริง
ตรวจจับหน้ากากผีตาโขน

เก็บภาพ label เทรน ทดสอบ — คะแนน = ความแม่นยำ / 10

1
รวบรวมรูปภาพ — เก็บภาพหน้ากากผีตาโขนให้หลากหลาย มุม แสง ขนาด อย่างน้อย 50 ภาพ
2
Label รูปภาพ — วาด bounding box + ระบุ class บน Roboflow Annotate (ใช้ Label Assist ช่วยได้)
3
จัดเตรียม dataset — แบ่ง train/val/test, เพิ่ม augmentation ให้เหมาะสม
4
เทรนโมเดล — Train บน Roboflow จาก pretrained MS COCO (YOLO11)
5
ทดสอบความแม่นยำ — วัด mAP บน test set บันทึกผลและ screenshot
🏆
คะแนน = mAP ÷ 10 — mAP 80% = 8 คะแนน · เน้น quality ของ annotation ให้ถูกต้องครบถ้วน
💡
Tips เพิ่ม mAP: label ให้แน่น bbox ชิดวัตถุที่สุด, เก็บภาพหลายมุม, ใช้ augmentation หลากหลาย, และมี training images อย่างน้อย 100+ ใบต่อ class
